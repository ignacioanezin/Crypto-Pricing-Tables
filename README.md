## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* npm
  ```sh
  npm install
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ignacioanezin/Crypto-Pricing-Tables
   ```
2. Init NPM
   ```sh
   npm init

3. Install node-sass
   ```sh
   npm install node-sass

4. Run the dev script
   ```sh
   npm run dev
